Sobre o Projeto

Fornecemos internet gratuita e ilimitada por meio de configurações em arquivos, garantindo inclusão digital para todos.

Como Importar um Arquivo no HTTP Injector

Passo 1: Obtenha o HTTP Injector

  • Certifique-se de que o aplicativo HTTP Injector já está instalado no seu dispositivo.

Passo 2: Obtenha o Arquivo de Configuração (.ehi)

  • Garanta que o arquivo de configuração (.ehi) seja compatível com seu provedor de internet.
  • Salve o arquivo na memória interna ou no cartão SD do seu dispositivo.

Passo 3: Abra o HTTP Injector

  • Localize e abra o aplicativo HTTP Injector no seu dispositivo.
  • Se for a primeira vez, aceite os termos de uso e conceda permissões solicitadas.

Passo 4: Importe o Arquivo de Configuração

  1. Na tela inicial, toque no ícone de Pasta no canto superior direito.
  2. Selecione Importar Configuração.
  3. Navegue até a pasta onde o arquivo .ehi está salvo e selecione-o.
  4. A configuração será carregada automaticamente.

Passo 5: Conecte-se à Internet

  1. Com a configuração carregada, toque no botão Iniciar ou Conectar.
  2. Se a configuração estiver correta, você verá mensagens de Conexão Bem-Sucedida.
  • Certifique-se de usar um arquivo

    Agora você sabe como importar um arquivo no HTTP Injector!

    • Como Importar um Arquivo no NPV Tunnel (Com o App Já Instalado)

    Passo 1: Obtenha o Arquivo de Configuração (.npv)

    • Certifique-se de que o arquivo de configuração (.npv) esteja salvo na memória interna ou cartão SD do seu dispositivo.

    Passo 2: Abra o NPV Tunnel

    • Localize o aplicativo NPV Tunnel no seu dispositivo e abra-o.

    Passo 3: Importe o Arquivo de Configuração

    1. Na tela inicial do app, toque no ícone de Três Pontos ou acesse o Menu (geralmente no canto superior direito).
    2. Selecione a opção Importar Configuração ou Load Config.
    3. Navegue até a pasta onde o arquivo .npv está salvo e toque nele para importar.
    4. A configuração será carregada automaticamente.

    Passo 4: Conecte-se à Internet

    1. Com a configuração carregada, clique no botão Iniciar ou Conectar.
    2. Se tudo estiver correto, uma mensagem de Conexão Estabelecida aparecerá.

    Agora você está pronto para usar o NPV Tunnel sem complicações!
